# MBECCUL Financial Dashboard

A professional microfinance dashboard for Mbengwi Cooperative Credit Union League.

## Demo Login
- Email: `admin@localhost.ac`
- Password: `react123`

## Setup
```bash
npm install
npm run dev
```

## Deploy to Vercel
```bash
git init && git add . && git commit -m "feat: MBECCUL capstone"
git remote add origin https://github.com/YOUR_USERNAME/mbeccul-dashboard.git
git push -u origin main
# Then connect repo on vercel.com
```

## Tech Stack
React 19 · Vite · React Router v6 · Context API + useReducer · Font Awesome · Recharts · localStorage
